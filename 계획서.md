**과목**: Classical Mechanics I (2026-1)
**제목**: Real-time Coupling of Discrete Grains and Porous Cloth Manifolds with Hysteretic Mass Transfer
**부제**: A JKR-informed cross-solver mass-transfer operator between MPM granular flow and XPBD cloth surface reservoir
**작성일**: 2026-04-25 (W2-pivot 반영, 3-agent adversarial verification 통과)

---

## 1. 한 단락 요약

본 연구는 **이산 입자(MPM granular flow)와 비-매니폴드 변형 천(XPBD cloth) 사이에서 가역적·hysteretic 질량 전이를 일으키는 cross-solver coupling operator**를 제안한다. 핵심은 *force가 아닌 constraint*로서의 JKR adhesion — Johnson-Kendall-Roberts 부착 모델의 pull-off energy를 XPBD constraint projection의 phase-transition threshold로 재정의하여, 입자가 천에 부착될 때는 MPM 자유도에서 제거되고 천 표면 reservoir(material-frame σ)의 dynamic mass-bearing state variable로 편입되며, 흔들기·관성에 의해 pull-off threshold를 초과하면 reservoir에서 다시 입자로 분리된다. 솔버 선택지(Eulerian MPM grid ↔ Lagrangian non-manifold XPBD surface)가 다른 두 표현 사이의 *비대칭적* 질량·운동량 전이를 RTX 4080 단일 GPU에서 실시간(≥30 fps)에 작동시키는 것이 시스템 기여다. 정량 평가는 mass + linear momentum dual conservation (각 < 1% drift over 10k frames). 정성 평가는 4개 ablation baseline 대비 차별이 가시적인 5개 demo.

## 2. 연구 배경

### 2.1 무엇이 *진짜로* 비어 있는가

10년에 걸쳐 그래픽스 입자 시뮬레이터는 두 갈래로 발전했다.

| 갈래 | 진화 | 도달점 |
|---|---|---|
| **MPM 계열** | Stomakhin 2013(snow) → MLS-MPM 2018 → MPM-FEM contact (BFEMP 2022, Dynamic Duo 2024) → real-time (PB-MPM 2024, XPBI 2024) | 입자-매니폴드 *contact* + 같은 솔버 내 multi-species (Mixture MPM 2021) |
| **Cloth 계열** | mass-spring → PBD/XPBD → IPC | thin-shell elastic + adhesive contact (Gascón 2010, Fang 2023) |

두 갈래가 만났을 때 풀린 문제 = **충돌·접촉, 같은 솔버 내 mixture, fabric에 액체 흡수.** 풀리지 않은 문제 = **두 솔버 사이의 *반복 가능한 양방향 질량 전이*와 reservoir 동역학.** 즉, 입자가 천에 들어가 천의 동역학적 상태 변수가 되었다가, 충분한 관성이 가해지면 다시 입자로 빠져나오는 cycle이 게임 framerate에서 작동하는 사례는 학술·산업을 통틀어 보고된 바가 없다.

### 2.3 본 연구가 *주장하지 않는* 것 (정직한 negation)

리뷰어 공격을 사전에 차단하기 위해 명시적으로 양보:

- "최초의 wet sand simulation"이라고 주장하지 않는다 — Rungjiratananon 2008, Wu 2023이 이미 함.
- "최초의 graphics JKR adhesion"이라고 주장하지 않는다 — DEM에서 30년 표준.
- "최초의 cloth-MPM coupling"이라고 주장하지 않는다 — Dynamic Duo 2024가 contact까지 함.
- "최초의 wet/dry sand transition"이라고 주장하지 않는다 — Wu 2023 saturation field가 함.
- "최초의 adhesive contact / hysteresis 모델"이라고 주장하지 않는다 — Fang 2023, Gascón 2010이 함.
- "최초의 material-frame surface field"라고 주장하지 않는다 — UV wet map은 production 표준.

본 연구가 *주장하는* 것: **위 모든 component를 위에서 정의한 5개 결정선 동시 만족하는 단일 cross-solver operator로 통합한 첫 시스템.**

### 2.4 왜 이 통합이 *비자명*한가 — 6개 본질적 어려움

| # | 어려움 | 왜 어려운가 | 본 연구의 접근 |
|---|---|---|---|
| A | **솔버 비대칭성** | MPM(Eulerian grid + Lagrangian particles) ↔ XPBD(Lagrangian non-manifold mesh)의 자료구조·시간 적분·constraint 처리 모두 다름 | Substep 단위에서 명시적 transfer phase 분리 |
| B | **JKR을 force가 아니라 constraint로** | 직접 force 적용 시 pull-off는 매우 stiff → 시간 스텝 폭발. 또한 force-only는 reservoir 자료구조와 결합 안 됨 | XPBD constraint projection의 phase-transition criterion으로 JKR pull-off energy 재정의 |
| C | **Non-manifold cloth surface 상의 reservoir** | 천은 fold·self-contact·double-sided 가능. UV에 σ를 저장해도 multi-layer accumulation·seam·projection ambiguity 처리 필요 | Per-triangle σ + double-sided mass + UV-cell aggregation, fold-aware projection |
| D | **이중 보존 (mass + linear momentum)** | Attach 시 입자 momentum이 vertex velocity에 어떻게 분배? Release 시 reservoir에서 정확히 그만큼 어떻게 차감? | (σ_j, p_σ_j) per-vertex pair, conservative split-and-commit operator |
| E | **Hysteresis가 진짜 JKR 곡선에서 나오게** | 단순 두 threshold(부착·이탈)는 휴리스틱. 진짜 JKR은 jump-to-contact radius와 pull-off necking radius가 다름 → energy curve 적분에서 자연스럽게 hysteresis | Maugis-Dugdale regularization으로 closed-form pull-off work 도출 |
| F | **Real-time 제약 하의 결정성** | Sorted reduction 없이 atomic 순서만으로는 bitwise reproducibility 불가능. 그러나 100% sorted reduction은 real-time 비용 폭발 | Aggregate reproducibility (statistical seed-determinism) + selective sorted reduction |

## 4. 방법

### 4.1 시스템 아키텍처

```
                          attach (JKR phase-transition)
   ┌──────────────────────┐                    ┌──────────────────────┐
   │  MPM granular field  │  ────── transfer ─▶│  Cloth UV reservoir  │
   │  (Eulerian grid +    │   commit phase     │  (per-triangle σ,    │
   │   Lagrangian particles) │                  │   per-vertex p_σ)    │
   │                      │ ◀── transfer ──── │   double-sided        │
   │                      │   release          │   non-manifold-aware │
   └──────────┬───────────┘                    └──────────┬───────────┘
              │                                            │
              │ gravity, contact (existing IPC/penalty)    │ cloth dynamics with
              │                                            │ effective inv_mass
              │                                            │   = 1/(m_v + Σ_T σ_T·w_vT)
              ▼                                            ▼
  ┌─────────────────────────────────────────────────────────────────────┐
  │                Conservation invariant (audited per substep)         │
  │     Σ m_p (active particles) + Σ σ_T·area_T = M0                    │
  │     Σ m_p v_p + Σ p_σ_j = P0                                        │
  └─────────────────────────────────────────────────────────────────────┘
```

### 4.2 Operator A — JKR-informed attach (constraint projection)

**Stick criterion** — 입자 i가 cloth triangle T (closest)에 거리 d, 충돌 시. Force가 아니라 *event predicate*로 정의:

$$
\text{Attach}(i, T) \iff E_n^{\text{kin},\,i} < W_{\text{adh}}^{\text{JKR}}(R_i, \gamma_T(\sigma, h)) \cdot \exp(-d/\lambda)
$$

- $E_n^{\text{kin},i} = \tfrac{1}{2} m_i (v_n^i)^2$: normal 방향 충돌 운동에너지
- $W_{\text{adh}}^{\text{JKR}}(R, \gamma) = \tfrac{15}{8\pi} \cdot \pi R^2 \cdot 2\gamma \cdot (3\pi\gamma R / 4 K)^{1/3}$: JKR pull-off work, 입자 반경 $R$, 표면 에너지 $\gamma$, 천 reduced modulus $K$ 의 닫힌 형식 (Johnson 1971; Maugis 1992)
- $\gamma_T(\sigma, h) = \gamma_0 + \beta \cdot h \cdot (1 - \sigma_T / \sigma_{\max})$: 표면 에너지가 humidity $h \in [0,1]$와 현재 saturation에 의존 → wet vs dry demo의 *물리적* 파라미터

**Attach 시 commit phase** (이벤트가 sort/compact된 후 한 번만 실행):

```
1. particles[i].active = 0   # MPM 도메인에서 제거
2. Δm = particles[i].mass
3. Δp = particles[i].mass · particles[i].vel  # linear momentum 동시 transfer
4. T(i)에 (b0, b1, b2) barycentric으로 σ, p_σ 분배:
   atomic_add(σ_T, Δm)
   atomic_add(p_σ_j_k, Δp · b_k) for k in {0,1,2}
5. Audit: dM_audit += Δm, dP_audit += Δp
```

### 4.3 Operator B — JKR-derived hysteretic detach

**Detach criterion** — vertex $j$ 의 effective inertial work이 *진짜 JKR pull-off necking energy*를 초과:

$$
\text{Detach}(j, k) \iff W_{\text{inertial}}(j, \Delta t) > W_{\text{pulloff}}^{\text{JKR}}(R_k, \gamma_T)
$$

- $W_{\text{inertial}}(j, \Delta t) = \int |\mathbf{a}_j(\tau)| \, d\tau$ over recent $\tau \in [t-\Delta t_w, t]$ (window)
- $W_{\text{pulloff}}^{\text{JKR}}$ = JKR jump-to-contact과 pull-off radius 차이에서 적분된 surface energy. **Hysteresis는 boolean이 아니라 두 radius 차이에서 자연 발생** (Maugis-Dugdale)

**Detach commit phase**:

```
1. Δm_release = MASS_PER_PARTICLE
2. atomic_sub(σ_T, Δm_release · b_k)  for k in {0,1,2}
3. Δp_release = (proportional vertex inertia from p_σ)
4. atomic_sub(p_σ_j, Δp_release)
5. Spawn particle in MPM at vertex pos ± normal·offset, vel = vertex.vel + Δp_release/Δm_release
6. Audit: dM_audit -= Δm_release, dP_audit -= Δp_release
```

### 4.4 Material-frame σ representation

| 설계 결정 | 이유 |
|---|---|
| **Per-triangle σ + per-vertex p_σ** | σ는 면적-종속, p_σ는 운동량 reservoir |
| **Double-sided** (`σ_T_front`, `σ_T_back`) | 천 양면에 다르게 누적 가능 (앞면이 모래밭 닿으면 앞만) |
| **UV-aware**: σ를 cloth UV 좌표에 위치-stable | World 좌표 vertex 방식의 swimming artifact 제거 |
| **Fold detection**: triangle pair angle < threshold이면 σ가 fold 안쪽으로 trapped | Reviewer가 "단순 wet map" 공격할 때 — fold-trapped sand는 wet map으로는 안 됨 |

### 4.5 Effective mass와 cloth dynamics 변경

$$
\tilde{m}_v = m_v^{\text{cloth}} + \sum_{T \ni v} \sigma_T \cdot w_{v,T}, \quad \mathbf{p}_v^{\text{eff}} = m_v \mathbf{v}_v + \mathbf{p}_{\sigma,v}
$$

**XPBD constraint solve에서 $\tilde{m}_v$ 사용** + position update가 끝난 후 $\mathbf{p}_v^{\text{eff}}$ 일관성 유지. Reviewer 검증용 metric: σ가 큰 vertex가 측정 가능하게 *더 처지고*, *swing period가 늘어남*.

### 4.6 Conservation 정리 (proof obligation)

> **Theorem (proof in §9):** 각 substep에서 Operator A와 B가 sort-then-commit 구조로 적용되면, 시스템 mass와 linear momentum은 단조 보존된다 (numerical roundoff 외):
> $$\frac{d}{dt}\left[\sum_{i: \text{active}} m_i + \sum_T \sigma_T A_T\right] = 0, \quad \frac{d}{dt}\left[\sum_i m_i \mathbf{v}_i + \sum_v \mathbf{p}_{\sigma,v}\right] = \mathbf{F}_{\text{ext}}$$

증명 스케치는 §9, 수치 검증은 [설명서.md](설명서.md) Step 9에서.

### 4.7 Real-time 결정성

- Same-seed aggregate determinism (입자별 결정 결과는 결정적, 그러나 atomic 순서 의존하는 σ 누적값은 ULP 수준 비결정성 인정)
- Bitwise determinism이 필요한 demo는 selective sorted reduction 활성화 (성능 -20% 추정)
- "Deterministic random sampling"으로 표현 약화 — bitwise determinism 주장 폐기

## 9. Conservation theorem (proof obligation)

**Lemma 1 (Operator A is mass-conservative).** Sort-then-commit 구조에서 attach event는 $(particles[i].mass) \to \sum_k \sigma_{T(i)}^k \cdot b_k$ 로 기여하므로 transfer당 $\sum_k b_k = 1$ 일 때 mass total이 보존된다.

**Lemma 2 (Operator A is linear momentum-conservative).** $\mathbf{p}_{\sigma,v} \mathrel{+}= m_i \mathbf{v}_i \cdot b_k$ 와 active set에서 $m_i \mathbf{v}_i$ 제거가 동시에 일어나면 total linear momentum 보존.

**Lemma 3 (Operator B reverse, by symmetry).** Same.

**Theorem 1.** Operator A and B가 sort-then-commit 형태이고 BVH 자체가 deterministic ordering을 주면, 시스템 invariant는 numerical roundoff 외 보존된다.

증명 작성과 수치 검증은 Week 9에 둘 다 완료.
